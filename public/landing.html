The engine

Not gut feelings. Actual math.

Eight quantitative models run simultaneously every time you analyze a prop. Here's what's under the hood.

Model 1
Bayesian Updating
Blends season averages with current game performance. Early game = season dominates. Late game = today dominates.
Model 2
Poisson Distribution
Treats scoring as a random process over continuous time. Gives true probability — not just a projection.
Model 3
Game State Engine
Blowout? Close game? 4th quarter? Each situation applies a different multiplier to the adjusted rate.
Model 4
Hot Hand Detection
Measures z-score deviation from season pace. Running hot gets a boost. Running cold gets discounted.
Model 5
Defensive Rating
Opponent defensive rating adjusts the baseline. Elite defense = harder. Weak defense = easier.
Model 6
Pace Projection
Current scoring pace extrapolated to final score. Powers the alt line EV scanner for game totals.
Model 7
Alt Line EV Scanner
Scans every available alt line and flags which ones have positive expected value vs the math's true probability.
Model 8
Kelly Criterion
Calculates optimal bet sizing based on the identified edge. Quarter-Kelly recommended for recreational bettors.
